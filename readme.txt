== Description ==
Redirect any "not found" (404) page to homepage with 301 (SEO) redirect code + Google Analytics (utm_source=404&utm_medium=Redirect&utm_campaign=404)

== Installation ==
1. Upload `pryc-wp-404-to-homepage-301-redirect` dir to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
